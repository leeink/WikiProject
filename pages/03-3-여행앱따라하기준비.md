## 네비게이션
이번 장에서는 다른 화면으로 이동해보는 것을 해볼 것이다.
다른 화면으로 이동하는 방법은 여러가지가 있는데 이번에는
Tab이라는 것으로 구현해볼 것이다. 실제 구현해보면 이동보다는
기존 화면을 다른 화면으로 갈아끼운다는 느낌이 들 것이다.

네비게이션을 위한 의존성을 추가해야 한다.
```kt
implementation("androidx.navigation:navigation-compose:2.9.8")
```

Finance.kt 파일이 있는 디렉토리에 Travel.kt 파일을 만든다.  
그리고 아래처럼 미리 이동할 화면들을 만든다.

```kt
@Composable
fun PopularScreen() {
    Column(
        modifier = Modifier.fillMaxSize(),
        verticalArrangement = Arrangement.Center,
    ){
        Text(text = "Popular Screen")
    }
}

@Composable
fun RecentScreen() {
    Column(
        modifier = Modifier.fillMaxSize(),
        verticalArrangement = Arrangement.Center,
    ){
        Text(text = "Recent Screen")
    }
}

@Composable
fun MostScreen() {
    Column(
        modifier = Modifier.fillMaxSize(),
        verticalArrangement = Arrangement.Center,
    ){
        Text(text = "Most Screen")
    }
}
```

네비게이션에 필요한 요소를 열거형으로 만든다.

```kt
enum class Tabs(
    val route: String,
    val label: String,
) {
    POPULAR("popular", "Popular"),
    RECENT("recent", "Recent"),
    MOST("most", "Most")
}
```

열거타입의 route로 네비게이션을 하고 label은
네비게이션을 하기 위한 탭 요소의 제목이다.
