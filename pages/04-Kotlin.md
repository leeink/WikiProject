## Kotlin Tour

이번 장에서는 코틀린(Kotlin)에 대해서 알아볼 것이다.
이번 장의 구성은 다음과 같다.

1. 기본 문법과 코틀린의 타입
2. 제어흐름과 형 변환
3. 함수
4. 클래스와 인터페이스

코틀린은 자바(Java)와 연관성이 깊은데 코틀린을 보다 더
잘 이해 하려면 자바를 우선 학습 해야 하지만 자바까지 학습을 한다면
이번 장이 너무 길어진다. 그래서 이번 장은 코틀린만 학습을 한다.

이번 장은 공식문서의 흐름 그대로 학습 과정을 이어나갈 것이다.

코틀린은 보통 다음과 같은 진입점에서 시작한다.

```kotlin
package com.example

fun main(){
    print("Kotlin Main Function")
}
```

프로젝트를 별도로 만들어서 학습해도 되지만
[웹 코틀린 연습공간](https://play.kotlinlang.org/)에서
진행해도 좋다. (https://play.kotlinlang.org/)
