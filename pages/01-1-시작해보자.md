# Jetpack Compose프로젝트 만들기

처음 시작하고 나면 app/src/main/com/example/(당신이 만든 앱 이름)/MainActivity.kt 파일을 열어보자.
열어보면 이런 내용이 있을 거다.
```kt
class MainActivity : ComponentActivity() {
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        enableEdgeToEdge()
        setContent {
            WikiAppTheme {
                App()
            }
        }
    }
}

@Composable
fun App() {
    Box(
        modifier = Modifier
            .fillMaxSize()
            .safeContentPadding()
            .background(MaterialTheme.colorScheme.background)
    ) {
        Text(
            text = "Hello. First Jetpack Compose App.",
            modifier = Modifier.align(Alignment.Center),
        )
    }
}

@Preview(showBackground = true)
@Composable
fun GreetingPreview() {
    WikiAppTheme {
        App()
    }
}
```

App()이라는 컴포저블 함수가 메인 화면을 구성하면서 화면이 보여지고 있다.

@Preview는 개발자가 구성한 화면을 바로바로 빠르게 확인할 수 있게 해주는 기능이다.
애뮬레이터만 활용하든 둘 다 활용하든 상관 없다. 필자는 @Preview를 잘 활용하지 않는 편이므로 앞으로 진행하면서 애뮬레이터만 사용할 거다.
