### 텍스트 스타일, 컬러 스타일 가져오기
이 작업은 피그마 디자인 파일을 보고 가져오지 말고 책에서 설명한 대로 따라하면 된다.  
프로젝트를 생성하면 MainActivity.kt파일이 있는 위치에 ui.theme라는 폴더가 있고 그 안에 Type.kt, Color.kt. Theme.kt가 있을 것이다. 우선 Type.kt를 열어보자.

```kt
val Typography = Typography(
    bodyLarge = TextStyle(
        fontFamily = FontFamily.Default,
        fontWeight = FontWeight.Normal,
        fontSize = 16.sp,
        lineHeight = 24.sp,
        letterSpacing = 0.5.sp
    ),
    bodyMedium = TextStyle(
        fontFamily = FontFamily.Default,
        fontWeight = FontWeight.Light,
        fontSize = 13.sp,
        lineHeight = 15.sp,
        letterSpacing = 0.sp
    ),
    titleLarge = TextStyle(
        fontFamily = FontFamily.Default,
        fontWeight = FontWeight.SemiBold,
        fontSize = 20.sp,
        lineHeight = 22.sp,
        letterSpacing = 0.sp
    ),
    titleMedium = TextStyle(
        fontFamily = FontFamily.Default,
        fontWeight = FontWeight.SemiBold,
        fontSize = 15.sp,
        lineHeight = 16.sp,
        letterSpacing = 0.sp
    ),
    titleSmall = TextStyle(
        fontFamily = FontFamily.Default,
        fontWeight = FontWeight.Medium,
        fontSize = 14.sp,
        lineHeight = 15.sp,
        letterSpacing = 0.sp
    )
)
```

위 코드는 필자가 이미 입력해놓은 것들이다. Typography라는 것을 통해서 텍스트 스타일이 계속 반복되는 것이 있으면 이 파일에 정의해놓고 가져다 쓰면 된다. Typography는 정해진 규격이 있는데

```kt
public constructor Typography(
    displayLarge: TextStyle = TypographyTokens.DisplayLarge,
    displayMedium: TextStyle = TypographyTokens.DisplayMedium,
    displaySmall: TextStyle = TypographyTokens.DisplaySmall,
    headlineLarge: TextStyle = TypographyTokens.HeadlineLarge,
    headlineMedium: TextStyle = TypographyTokens.HeadlineMedium,
    headlineSmall: TextStyle = TypographyTokens.HeadlineSmall,
    titleLarge: TextStyle = TypographyTokens.TitleLarge,
    titleMedium: TextStyle = TypographyTokens.TitleMedium,
    titleSmall: TextStyle = TypographyTokens.TitleSmall,
    bodyLarge: TextStyle = TypographyTokens.BodyLarge,
    bodyMedium: TextStyle = TypographyTokens.BodyMedium,
    bodySmall: TextStyle = TypographyTokens.BodySmall,
    labelLarge: TextStyle = TypographyTokens.LabelLarge,
    labelMedium: TextStyle = TypographyTokens.LabelMedium,
    labelSmall: TextStyle = TypographyTokens.LabelSmall
)
```

위와 같은 정해진 것들만 사용할 수 있다. 기본값들이 들어가있으나 필자처럼 정해진 규격 안에서만 변경이 가능하긴 하다.
필자는 많은 것을 사용하진 않을거라 5개만 따로 설정하였다. 이 작업을 왜 하는지 궁금해 할 수 있는데 화면 하나를 만들면 텍스트가 생각보다 많은데 그때마다 하나하나 폰트 크기 정하고 굵기 정하고 자간, 행간 정하고 있으면 정말 피곤하다. 그리고 디자인은 어느 정도 보기 좋은 통일성이 필요한데 이건 디자인에 관한 이야기이니 여기까지만 이야기하고 넘어가도록 한다.
